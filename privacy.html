Privacy / GDPR Information

This page exists to provide transparency for a student front-end project.

What this site is

  • This website is a student project created to learn Bootstrap and deployment basics.
  • Displayed cryptocurrency “prices” are fake demo values and are not market data.

Personal data

  • No personal data is collected or stored on a server.
  • There is no analytics and no tracking cookies implemented by this project.

Contact form

  • If you type anything into the contact form, it is processed only in your browser (client-side).
  • No form submissions are sent to a backend, and the project does not retain messages.

Third-party resources

The site loads these libraries from CDNs:

  • Bootstrap CSS/JS bundle
  • Font Awesome icons
CDNs may log standard request metadata (like IP address) as part of normal web hosting operations.

Contact

For assignment purposes, you can reach the project owners via the on-page contact form (demo) on the homepage.
